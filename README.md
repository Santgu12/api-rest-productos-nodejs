# API REST de Productos

API REST desarrollada con Node.js, Express y MySQL para la gestión de productos.

## Tecnologías

- Node.js
- Express
- MySQL
- dotenv
- cors

## Funcionalidades

- Obtener todos los productos
- Obtener producto por ID
- Crear productos
- Actualizar productos
- Eliminar productos

## Instalación

```bash
npm install
```
